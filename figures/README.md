This directory contains Jupyter notebooks and data files needed to reproduce the figures in the NanoMAP manuscript. If you are only interested in running NanoMAP clustering and analysis you can ignore or delete this directory.
# Additional large files needed
Several files needed to reproduce these figures are too large to store on GitHub. If you would like to reproduce the figures you can download them from Zenodo (doi: 10.5281/zenodo.18329868) or the VDJserver (study ID: PRJNA1466709)
\
Once you have the files, transfer them into the `large_files` directory here before running the associated Jupyter notebooks.
